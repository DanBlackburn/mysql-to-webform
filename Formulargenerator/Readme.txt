----- Generell
Entwicklungsumgebung: Visual Studio 2013 Ultimate mit dem Plugin "PHP Tools for Visual Studio" (PHP 5.4)
Die Seite benötigt JavaScript um korrekt zu funktionieren,

----- Testen
Browser: Internet Explorer 11 (für die Analyse von HTML-Elementen) und Google Chrome  (für das Testen der Eingabevalidierung)

Datenbankname: formulargenerator
Datenbanktabellen:
- Mixed (Gemischte Datentypen)
- Non_input (Alle Felder die nicht einem Input-Element resultieren)
- Input_required (Alle Felder die einem Input-Element resultieren, alle Felder erforderlich)
- Input_not_required (Alle Felder die einem Input-Element resultieren, alle Felder nicht erforderlich)
