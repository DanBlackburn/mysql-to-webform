Per column:
- Null/ not Null
- Default value

Examples:
Männlich, BIT                              -> Input (Checkbox)            
Geschlecht, ENUM('Männlich', 'Weiblich')   -> Input (Radio)       | Conflict with select
Vorname, VARCHAR(35)                       -> Input (35)          | Missing type (date, number, ..)
Beschreibung, TEXT(256)                    -> Textarea (256)      
Bewertung, ENUM('Scheisse', 'Ok' ,'Gut')   -> Select              | Conflict with radiobuttons
Password, VARCHAR(256)                     -> Input (Pw, 256)     | Missing password attribute

- BIT      -> Input (Checkbox)
- ENUM     -> Input (Radio)/ Select
- VARCHAR  -> Input ([Custom])
- TEXT     -> Textarea

Conflicts:
- ENUM: Radiobutton/ Select   -> postfix in name (_r/ _s)
- VARCHAR: type, password     -> postfix in name ([PLACEHOLDER])
